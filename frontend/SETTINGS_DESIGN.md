# 系统设置页面设计文档

## 设计概述
基于 `docs/genpulse_ux_ui_design_guide/_3/code.html` 的设计规范，实现了现代化的暗色主题系统设置页面。

## 设计特点

### 1. 视觉设计
- **暗色主题**: 使用Material Design 3的暗色调色板
- **现代化UI**: 玻璃态效果、渐变背景、微交互
- **响应式布局**: 适应不同屏幕尺寸
- **图标系统**: 使用Material Symbols图标

### 2. 色彩系统
基于设计文件中的调色板：
- **主色**: `#c0c1ff` (紫色)
- **主容器**: `#5b5fff` (深紫色)
- **表面颜色**: `#131318` (深灰背景)
- **表面容器**: `#1f1f25` (卡片背景)
- **文字颜色**: `#e4e1e9` (浅灰文字)
- **轮廓颜色**: `#908fa1` (灰色边框)

### 3. 字体系统
- **标题字体**: Inter (600-700字重)
- **正文字体**: Inter (400-500字重)
- **代码字体**: JetBrains Mono (400-500字重)
- **图标字体**: Material Symbols Outlined

## 页面结构

### 1. 通用设置 (General Settings)
采用Bento网格布局，包含：
- **界面主题**: 浅色/深色/跟随系统切换按钮
- **语言设置**: 简体中文/English (US) 下拉选择
- **系统日志级别**: DEBUG/INFO/WARN/ERROR 下拉选择

### 2. 模型提供商配置 (Model Configuration)
采用堆叠式列表布局，包含：
- **OpenAI**: 已连接状态，支持GPT模型选择
- **Anthropic Claude**: 已连接状态，支持Claude模型选择
- **Google Gemini**: 未配置状态，需要配置API密钥

**状态指示器**:
- 🟢 已连接 (绿色)
- 🟡 未连接 (黄色)
- 🔴 未配置 (红色)

### 3. 全局Agent角色模板 (Agent Role Templates)
采用玻璃态卡片布局，包含：
- **Orchestrator**: 任务路由与分解角色
- **Architect**: 系统设计与架构角色
- **Frontend Dev**: UI生成与逻辑角色

**每个角色卡片包含**:
- 角色图标和标题
- 活跃状态指示器
- 分配的模型选择器
- 系统提示词预览和编辑功能

### 4. 高级设置 (Advanced Settings)
包含开关选项：
- **实验性功能**: 启用开发中的新功能
- **匿名遥测数据**: 发送匿名使用数据
- **开发者模式**: 显示开发者工具

## 技术实现

### 1. 状态管理
```typescript
interface SettingsData {
  general: {
    theme: 'light' | 'dark' | 'auto';
    language: string;
    logLevel: 'debug' | 'info' | 'warn' | 'error';
  };
  modelProviders: ModelProvider[];
  agentRoles: AgentRole[];
  advanced: {
    enableExperimental: boolean;
    enableTelemetry: boolean;
    developerMode: boolean;
  };
}
```

### 2. 组件结构
- **SettingsView**: 主容器组件
- **标签导航**: 使用按钮组实现
- **设置卡片**: 使用Card组件包装
- **模型列表**: 使用Flex布局实现
- **角色卡片**: 使用Grid布局实现

### 3. 样式系统
- **Tailwind CSS**: 基础样式
- **自定义CSS**: 设计系统特定样式
- **CSS变量**: 主题颜色变量
- **动画效果**: 平滑过渡和微交互

## 响应式设计

### 断点设计
- **移动端**: < 640px (单列布局)
- **平板端**: 640px - 1024px (两列布局)
- **桌面端**: > 1024px (多列布局)

### 布局适配
- 通用设置: 移动端单列，桌面端三列
- 模型列表: 始终单列，自适应宽度
- 角色卡片: 移动端单列，桌面端两列

## 使用说明

### 访问设置
1. 点击侧边栏的Settings图标 (⚙️)
2. 系统设置页面将显示在主内容区域

### 修改设置
1. 点击顶部标签切换到相应设置类别
2. 修改表单中的设置值
3. 浮动保存按钮将自动出现
4. 点击"保存配置"按钮保存设置

### 测试连接
1. 在模型提供商配置中点击"测试连接"
2. 系统将验证API连接状态
3. 状态指示器将实时更新

### 配置API密钥
1. 点击未配置提供商的"配置密钥"按钮
2. 在弹出的对话框中输入API密钥
3. 系统将自动测试连接并更新状态
