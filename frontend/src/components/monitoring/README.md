# 监控仪表盘组件

根据《WBS 任务分解.md》文档中第四阶段4.3监控仪表盘开发的要求，实现了完整的监控仪表盘系统。

## 实现的功能组件

### 1. Agent状态仪表盘 (AgentStatusDashboard)
- 卡片式展示所有Agent的实时状态与进度
- 包含Agent类型、当前任务、进度条、时间统计等信息
- 性能统计图表和Agent分布饼图
- 实时更新Agent状态和性能指标

### 2. 执行时间线视图 (ExecutionTimeline)
- 甘特图形式展示各Agent的执行时间段
- 基于Trace数据的可视化时间线
- 支持播放、暂停、快进、快退等控制
- 缩放和时间范围选择功能
- 事件详情查看和筛选

### 3. 思维链面板 (ThoughtChainPanel)
- 实时流式展示当前Agent的推理过程
- 支持内部推理和解决方案制定两种思维类型
- 代码生成内容的特殊展示
- 播放控制、搜索和过滤功能
- 思维链的展开/收起和复制功能

### 4. 工具调用日志表格 (ToolLogsTable)
- 表格形式展示工具名、参数、返回值、耗时
- 支持按状态、工具类型、时间排序
- 详细的工具调用信息和参数查看
- 导出和复制功能
- 工具类型分布统计

### 5. 终端输出面板 (TerminalOutputPanel)
- 内嵌终端模拟器，显示命令执行输出
- 支持日志级别过滤和搜索
- 自动滚动和手动控制
- 系统资源监控（CPU、内存、磁盘、网络）
- 命令历史记录和重新执行

### 6. 文件变更Diff预览 (FileDiffPreview)
- 展示Agent创建/修改的文件diff
- 支持统一视图和分屏视图
- 行号显示和空白字符控制
- 文件类型过滤和变更类型筛选
- 文件树结构预览

### 7. 成本仪表盘 (CostDashboard)
- 统计Token消耗与费用估算
- 按成本类型（LLM、API、存储、计算）分组
- 时间序列成本图表和饼图分布
- 代理成本分析和优化建议
- 多货币支持和详细交易记录

### 8. 自进化监控面板 (EvolutionMonitor)
- 展示Skill生成事件、记忆检索命中率等
- 进化事件类型分布和趋势分析
- 效率提升、Token节省、时间节省统计
- 学习模式和优化机会分析
- 成就系统和成长预测

## 使用说明

### 启动监控
1. 在侧边栏点击"监控仪表盘"
2. 系统自动开始数据采集
3. 启用自动刷新以获取实时数据

### 基本操作
- **筛选**：使用顶部筛选器按类型、状态、时间过滤
- **搜索**：在搜索框中输入关键词查找特定内容
- **排序**：点击表格标题按列排序
- **查看详情**：点击任何项目查看详细信息
- **导出数据**：使用下载按钮导出当前视图数据

### 高级功能
- **时间线控制**：使用播放控制条回放执行过程
- **思维链播放**：按顺序播放Agent的思考过程
- **终端命令**：查看和重新执行历史命令
- **成本分析**：查看详细成本分解和优化建议
- **进化监控**：跟踪Agent的学习和优化过程

## 集成说明

### 后端要求
监控仪表盘需要后端提供以下API接口：
- `GetAgentsStatus()`：获取Agent状态
- `GetTimelineEvents()`：获取时间线事件
- `GetThoughts()`：获取思维链数据
- `GetToolLogs()`：获取工具调用日志
- `GetTerminalOutput()`：获取终端输出
- `GetFileDiffs()`：获取文件差异
- `GetCostMetrics()`：获取成本指标
- `GetEvolutionEvents()`：获取进化事件
- `GetMonitoringStats()`：获取监控统计
- `SendIntervention()`：发送干预指令

### 数据格式
所有数据接口应返回JSON格式数据，具体格式参考TypeScript类型定义。

## 故障排除

### 常见问题
1. **数据不更新**：检查自动刷新设置和网络连接
2. **图表不显示**：检查数据格式和图表配置
3. **性能问题**：减少同时显示的组件数量
4. **内存泄漏**：确保组件正确清理定时器和监听器
