# 历史记录与回放功能实现

## 概述
根据WBS任务分解中的4.4节要求，实现了历史记录与回放功能，包含两个主要部分：
1. **执行历史列表** - 展示所有历史执行记录，支持筛选和搜索
2. **执行回放功能** - 加载历史Trace，按时间轴回放执行过程

## 实现的功能

### 1. 后端服务 (`internal/monitoring/history/`)
- **`service.go`** - 历史记录核心服务
  - `ExecutionRecord` 结构体：存储执行记录信息
  - `ReplayState` 结构体：管理回放状态
  - `HistoryService` 类：提供CRUD操作、查询、统计功能
  - 支持执行记录的创建、更新、查询、删除
  - 支持回放的开始、暂停、恢复、停止、跳转、速度控制

- **`wails_bridge.go`** - Wails桥接器
  - 提供前端可调用的API接口
  - 事件推送机制，实时更新回放状态
  - 与前端的事件订阅机制

### 2. 前端组件 (`frontend/src/components/monitoring/`)
- **`HistoryList.tsx`** - 历史记录列表组件
  - 展示所有执行记录的表格视图
  - 支持按状态、标签、时间范围筛选
  - 支持按名称、描述搜索
  - 支持按时间、时长、Token使用量等排序
  - 分页显示，每页20条记录
  - 统计信息展示面板
  - 操作按钮：回放、导出、删除

- **`ReplayPlayer.tsx`** - 回放播放器组件
  - 时间线视图，显示所有Span
  - 播放控制：开始、暂停、停止、跳转
  - 速度控制：0.25x、0.5x、1x、2x、4x、8x
  - 进度条，支持拖拽跳转
  - 当前Span详细信息展示
  - 全屏模式支持
  - 实时状态更新

### 3. API集成
- **`frontend/src/services/api.ts`** - 扩展API服务
  - 添加历史记录相关方法：`getHistoryRecords`、`getHistoryStatistics`等
  - 添加回放相关方法：`startReplay`、`controlReplay`、`getReplayData`等
  - 事件订阅机制：`subscribeToReplayEvents`
  - 模拟数据支持，用于开发和测试

- **`app.go`** - 主应用集成
  - 添加历史记录服务初始化
  - 实现所有历史记录相关的Wails绑定方法
  - 集成到应用启动流程

### 4. 监控仪表盘集成
- **`MonitoringDashboard.tsx`** - 更新监控仪表盘
  - 添加"History"标签页
  - 集成历史记录列表组件
  - 支持从历史记录直接启动回放
  - 回放时显示专用播放器界面

## 数据结构

### ExecutionRecord 执行记录
```go
type ExecutionRecord struct {
    ID           string                 // 记录ID
    TraceID      string                 // Trace ID
    PipelineID   string                 // 管道ID
    Name         string                 // 执行名称
    Description  string                 // 描述
    Status       string                 // 状态：running, completed, failed, cancelled
    StartTime    time.Time              // 开始时间
    EndTime      time.Time              // 结束时间
    Duration     time.Duration          // 执行时长
    AgentCount   int                    // 参与的Agent数量
    ToolCallCount int                   // 工具调用次数
    TokenUsage   int                    // Token使用量
    CostEstimate float64                // 成本估算
    Metadata     map[string]interface{} // 元数据
    Tags         []string               // 标签
    CreatedAt    time.Time              // 创建时间
    UpdatedAt    time.Time              // 更新时间
}
```

### ReplayState 回放状态
```go
type ReplayState struct {
    RecordID          string                 // 关联的记录ID
    TraceID           string                 // Trace ID
    Status            string                 // 状态：initializing, playing, paused, completed, error
    CurrentTime       time.Time              // 当前回放时间
    StartTime         time.Time              // 开始时间
    EndTime           time.Time              // 结束时间
    PlaybackSpeed     float64                // 播放速度
    CurrentSpanIndex  int                    // 当前Span索引
    TotalSpans        int                    // 总Span数
    Progress          float64                // 进度百分比
    Metadata          map[string]interface{} // 元数据
}
```

## 使用示例

### 1. 创建执行记录
```go
record := &history.ExecutionRecord{
    Name:        "用户认证系统开发",
    Description: "完整的JWT认证系统实现",
    PipelineID:  "pipeline-001",
    Status:      "completed",
    StartTime:   time.Now().Add(-2 * time.Hour),
    EndTime:     time.Now().Add(-1.5 * time.Hour),
    Duration:    30 * time.Minute,
    AgentCount:  4,
    ToolCallCount: 42,
    TokenUsage:  12500,
    CostEstimate: 0.85,
    Metadata: map[string]interface{}{
        "primary_agent": "Orchestrator",
        "project_type":  "web",
    },
    Tags: []string{"production", "authentication"},
}
```

### 2. 查询历史记录
```javascript
// 前端调用
const [records, total] = await api.getHistoryRecords({
    searchText: "认证",
    statuses: ["completed"],
    tags: ["production"],
    sortBy: "start_time",
    sortOrder: "desc",
    limit: 20,
    offset: 0
});
```

### 3. 开始回放
```javascript
// 开始回放，速度1.0x
const state = await api.startReplay(recordId, 1.0);

// 订阅回放事件
api.subscribeToReplayEvents(recordId, (event, data) => {
    if (event === 'replay:progress') {
        console.log(`回放进度: ${data.state.progress}%`);
    } else if (event === 'replay:ended') {
        console.log('回放结束');
    }
});

// 控制回放
await api.controlReplay(replayId, 'pause');
await api.controlReplay(replayId, 'seek', { progress: 50 }); // 跳转到50%
await api.controlReplay(replayId, 'speed', { speed: 2.0 });  // 2倍速
```

## 技术实现细节

### 数据库架构
```sql
-- 执行记录表
CREATE TABLE execution_records (
    id TEXT PRIMARY KEY,
    trace_id TEXT NOT NULL,
    pipeline_id TEXT,
    name TEXT NOT NULL,
    description TEXT,
    status TEXT NOT NULL,
    start_time DATETIME NOT NULL,
    end_time DATETIME,
    duration INTEGER,
    agent_count INTEGER NOT NULL DEFAULT 0,
    tool_call_count INTEGER NOT NULL DEFAULT 0,
    token_usage INTEGER NOT NULL DEFAULT 0,
    cost_estimate REAL,
    metadata TEXT,  -- JSON格式
    tags TEXT,      -- JSON数组格式
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 全文搜索虚拟表
CREATE VIRTUAL TABLE records_fts USING fts5(
    id UNINDEXED,
    name,
    description,
    tags,
    content='execution_records',
    content_rowid='rowid'
);
```

### 性能优化
1. **索引优化**：
   - 主键索引：`id`
   - 查询索引：`trace_id`, `pipeline_id`, `status`, `start_time`, `end_time`

   - 使用SQLite FTS5实现高性能全文搜索

3. **查询优化**：

## 使用示例

### 完整工作流程
```go
// 1. 初始化服务
storage, _ := history.NewHistoryStorage("history.db")
traceStore, _ := storage.NewTraceStore("traces.db")
traceAdapter := history.NewTraceStoreAdapter(traceStore)
service := history.NewHistoryServiceWithStorage(storage, traceAdapter)

// 2. 创建执行记录
record := &history.ExecutionRecord{
    Name: "用户认证系统",
    TraceID: "trace-123",
    Status: "completed",
    // ... 其他字段
}
service.CreateRecord(ctx, record)

// 3. 查询和筛选
query := history.ExecutionQuery{
    SearchText: "认证",
    Statuses: []string{"completed"},
    Tags: []string{"security"},
    Limit: 20,
    SortBy: "start_time",
    SortOrder: "desc",
}
records, total, _ := service.QueryRecords(ctx, query)

// 4. 启动回放
state, _ := service.StartReplay(ctx, record.ID, 1.0)

// 5. 控制回放
service.ControlReplay(ctx, state.RecordID, "pause", nil)
service.ControlReplay(ctx, state.RecordID, "seek", map[string]interface{}{
    "progress": 50.0,
})

// 6. 获取回放数据
spans, _ := service.GetReplayData(ctx, state.RecordID, 0, 100)

// 7. 获取统计信息
stats, _ := service.GetStatistics(ctx, startTime, endTime)
```
