## 使用示例

### 通过UI使用Agent
1. 打开Agent管理界面
2. 选择"全栈开发助手"Agent
3. 输入任务："创建一个Go HTTP服务器，提供REST API"
4. 配置参数：端口=3000，项目名称=demo-server
5. 点击执行，查看生成的Go代码

### 通过API使用Agent
```go
// 创建Agent
config := agents.AgentConfig{...}
agent, _ := agents.NewFullstackDeveloperAgent(config, modelAdapter, toolRegistry, flowEngine)

// 执行任务
result, _ := agent.Execute(ctx, "创建HTML产品页面", map[string]interface{}{
    "title": "产品展示",
    "content": "产品描述内容",
})
```
